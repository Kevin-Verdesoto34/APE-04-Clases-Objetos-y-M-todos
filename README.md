# APE-04-Clases-Objetos-y-Metodos

Desarrollo de un sistema básico de gestión académica en los lenguajes C++ y Java mediante el paradigma de Programación Orientada a Objeto
(POO), que permita registrar de forma validada la información y calificaciones de los estudiantes de la asignatura Algoritmos y Lógica de
Programación, automatizando el cálculo de sus promedios y la determinación de su estado académico.

Lenguajes Utilizados:
C++
Java
